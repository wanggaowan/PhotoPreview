# PhotoPreview
仿微信朋友圈图片预览。此库参考 [PhotoViewer](https://github.com/wanglu1209/PhotoViewer)，并在此基础上做了修改，修复内存泄漏问题，优化动画,现在打开关闭更加流畅。

效果请下载[demoApk](/app-debug.apk)查看

#### Proguard
无需添加任何混淆规则，可直接混淆
